# Aura Countdown - 项目介绍

## 1. 项目简介

**Aura Countdown** 是一款基于 Electron 和 Vue 3 构建的 macOS 状态栏倒计时应用。它旨在提供一个轻量、美观且不打扰用户的倒计时管理工具。用户可以快速创建、查看和管理多个倒计时事件，并支持将重要事件置顶显示在 macOS 状态栏上。

## 2. 核心功能

*   **状态栏驻留**：应用常驻 macOS 状态栏，点击图标即可快速下拉查看所有倒计时。
*   **倒计时管理**：支持创建、编辑和删除倒计时事件。
*   **置顶显示**：支持将任意一个“进行中”的事件置顶，置顶后状态栏将直接显示该事件的剩余天数，方便随时查看。
*   **进度可视化**：每个倒计时卡片都带有进度条，直观展示时间流逝情况。
*   **多屏幕支持**：智能识别鼠标所在屏幕，点击状态栏图标时，窗口会在当前屏幕正确弹出。
*   **现代 UI 设计**：采用清爽、现代的卡片式设计，支持自定义图标，界面优雅美观。

## 3. 适用场景

*   **项目截止日期**：追踪项目里程碑和交付时间。
*   **重要纪念日**：记录生日、周年纪念等特殊日子。
*   **考试/活动倒计时**：为即将到来的考试、假期或活动做准备。
*   **日常任务**：管理短期任务的截止时间。

## 4. 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev:electron
```

### 打包构建
```bash
npm run build:mac
```

## 5. 项目结构概览

*   `src/main.cjs`: Electron 主进程代码，负责窗口管理、托盘逻辑和数据持久化。
*   `src/App.vue`: Vue 3 渲染进程代码，负责 UI 展示和交互逻辑。
*   `src/preload.cjs`: 预加载脚本，用于主进程和渲染进程之间的安全通信。
*   `countdowns.json`: 用户数据存储文件（位于用户数据目录）。
